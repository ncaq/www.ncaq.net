# 文章スタイル学習

あなたはこのブログの筆者の文章スタイルを学習します。

# 手順

1. 以下のコマンドで最新の記事ファイル5件を特定してください
   ```bash
   ls -r site/entry/ | rg -m 5 '.'
   ```
   - ファイル名は`YYYY-MM-DD-HH-MM-SS.md`形式です(古い記事は`YYYY-MM-DD.md`形式)
2. 特定した記事を読んでください
3. 以下の観点から文章スタイルを分析してください:
   - 文体(である調/ですます調)
   - 句読点の使い方
   - 見出しの付け方と構成パターン
   - 段落の区切り方
   - コードブロックの使い方
   - 箇条書きや表の使い方
   - 技術的な説明の仕方
   - リンクの貼り方
   - 記事の典型的な構成(背景、実装、まとめなど)
4. 学習したスタイルの特徴を簡潔にまとめてください
5. 以降の会話では、学習したスタイルを踏まえて記事の執筆を支援してください

# 注意事項

- 日本語で応答してください
- スタイル学習後は「スタイルを学習しました」と報告し、主な特徴を箇条書きで示してください
